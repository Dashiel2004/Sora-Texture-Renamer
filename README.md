# Sora Outfit Generator
A tool made to coincide with my Changing Outfits Midgame mod for Super Smash Bros. Ultimate. This is solely made for Sora's color 2, 4, 6, and 8. (His Kingdom Hearts 2 and 3 slots) for now.

This tool includes all of Sora's kh2 and kh3 formchanges (Except double, anti, and rage), the tool allows you to choose which form you want for each outfit slot (1-5), the textures are included in the asset folder, all you have to do is select the forms you want for the outfit slot, and you can choose which slot you want these for, once you click rename textures, the program will automatically rename the textures and put them in the output/(slot) folder, once the files have been put in the output its as simple as moving the files to the template provided from the changing outfits mod.

Lets say for example you choose a slot c01 and then choose your outfits and then hit the rename textures button, in the output folder, find the c01 folder and take all the textures in the folder and move them to the template folder/fighter/trail/model/body/c01 folder and replace any files it asks about. as simple as that, now you can boot up Ultimate and change forms at will!

Requires Visual C++ v14 Redistributable: https://learn.microsoft.com/en-us/cpp/windows/latest-supported-vc-redist?view=msvc-170#latest-supported-redistributable-version
